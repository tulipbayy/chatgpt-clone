# ChatGPT Clone

A mini-project starter with a React frontend and Express backend.

## Setup

1. Install dependencies:

```bash
npm run install:all
```

2. Create the backend environment file:

```bash
cp backend/.env.example backend/.env
```

3. Add your OpenAI API key to `backend/.env`.

4. Run both apps:

```bash
npm run dev
```

The frontend runs at `http://localhost:5173` and the backend runs at `http://localhost:5001`.
